# This Is Backend Server MernStack
simple app for beginners like me
# Requirements
* `node & npm`
# Dependencies 
* `mongoose`
* `bcrypt`
* `dotenv`
* `express`
* `jsonwebtoken / jwt`
* `validator`
# How To Run 
* `npm install`
* `configure your .dotenv becausee need secret`
* `configure your port recommended : 4000`
* `npm run build || node index.js`
